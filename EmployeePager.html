Project Description

Inspired by employee chat services like Slack and Microsoft team, this web app allows users create accounts, live chat, edit and delete old chats, send images videos and track group messages. Additionally, the application uses local storage to allow users to select a theme and securly remember the their login token. User Authentication and data storage is done through Stream.io. This was a full stack application so, I had to create Api routes to maintain constant communication between stream.io and the users instance. Finally, the frontend was created with css.
